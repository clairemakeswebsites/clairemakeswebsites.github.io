Cocktail Picker

Cocktail picker web application built using React and Create React App. Used starter code to begin the project and the Cocktail DB API to fetch cocktail information. Then connected the site to a github repository for continuous development.


Used React-Router-Dom to manage different pages making it easy for the user to find the information they are looking to find. Added a search function to the app so the user can search for a specific cocktail. Implemented a loading component while waiting for the cocktail information to be fetched from the API and displayed on the webpage.
