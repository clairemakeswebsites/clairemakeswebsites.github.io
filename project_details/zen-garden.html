CSS Zen Garden

This is my take on CSS Zen Garden. I chose the theme gradients to stretch my CSS skills in creating a unique visually appealing design. I used gradients as both a major design element and accent elements. This theme created the challenge of maintaining enough contrast on the webpage, as well as creating accent elements without changing the HTML.


Since this project was created over 10 years ago practices around accessibility and designing for accessibility have expanded to make websites more accessible for all users. I made a few changes to the HTML to make the webpage more accessible. I changed div elements to semantic HTML and created a main section to create landmarks for users who use screen readers or adaptive technology to use the web.
