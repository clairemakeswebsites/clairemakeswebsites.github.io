Photographer Biography

This project was part of an assignment for a web design class and it was to choose an artist and create a website with their background and showcase art. The site had to be hand coded, responsive, had to have at least 3 web pages linked together with the use of a menu, embed at least one video, include headings, and paragraphes using semantic HTML, and had to implement design principles to look aesthetically pleasing. For this project I chose the photographer, Anna Atkins I started the design process using figma to create a layout for each screen size.


For the menu, I used a responsively designed navigation bar with javascript and css media queries. On mobile screen sizes the navigation is a slide out menu, that then becomes a navigation bar at the top of each webpage at tablet screen size and larger. Then used Javascript to create an image carousel showcasing her two books, British and Foreign Ferns, and British Algae with captions for each image.
