Cards

Little Details challenge from codepen. Challenge focused on finding easy to add little details to make the design look aesthetically pleasing, as well as supporting functionality. The challenge gave starter code with just text and images.


Added subtle details using CSS to make the design support functionality. Each card has a box-shadow, matching the background of the image, to make each card pop and distinguish it from the other cards. Each card flips when hovered on to give more information about each card. Adding those details made each card unique while maintaining the general styling, making the page look cohesive and easy to understand while making each card distinct from the others.
