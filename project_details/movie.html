Movie Database

Using Create React App, The Movie DB AP, and starter files created a movie database. Used the Movie Database API to fetch information about recently released movies then took that data and created a web application to display the movies and their information. Used React-Router- to handle routing between different pages, each movie had its own page as well as a not found page for movies that were not found.


Used hooks to access React features for added functionality in the web application. Added a search function and useState to manage state so the user could search for a movie.The web application styles were created with Styled Components to create consistent styling across the web application as well as making it responsive and accessible to a variety of users.
