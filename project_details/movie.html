Movie Database

Built a movie database web application using React, Create React App, href=”https://www.themoviedb.org/documentation/api”>the Movie DB API, and these starter files. The web app displays movie information fetched from the Movie DB. Users can find current movies displayed in the web application and search for a specific movie. Then click on a title to find out more information about that movie.


The application also uses React Router to handle routing between the main page and individual movie pages. Took advantage React hooks to implement features such as searching and to fetch movie data from the Movie DB. Used styled components to easily create consistent styling across the web application.
