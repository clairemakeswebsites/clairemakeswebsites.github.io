Space Tourism

Space tourism challenge from FrontEnd Mentor and Scrimba. FrontEnd Mentor provided the starter code and design files, and Scrimba had tutorial videos. The challenge was to create a multipage website as close to the design files as possible. It had to include the following features- the user should be able to view the optimal for each of the website’s pages depending on the user’s screen size, see the hover states for all interactive elements on each page, and view each page and toggle between the tabs to see new content.


I began building this site by creating a design system that could quickly and easily apply styles to different elements across the site. I used flexbox, grid, and media queries to make the website responsive so it was optimized at different screen sizes. Lastly I completed the website by adding JavaScript functionality. The user is able to smoothly tab through the content on the website
