Travel Blog

Project from Scrimba Learn React course. The course provided starter code, and a figma design. The goal of the project was to create a travel blog using the information from the provided data.js file and to create a design based on the provided figma file.


Starting the project I divided the parts of the webpage into components, creating separate components for the header and the travel information. I took the data from the data.js file and used the map() function to iterate through the data in that file and then display the information in the webpage. Then using CSS to create a similar design to the one from figma.
